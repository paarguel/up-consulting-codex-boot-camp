# Privacy Review Checklist

Use this before putting real work material into Codex.

- [ ] Does this include real names?
- [ ] Does this include children, parishioners, clients, donors, or staff issues?
- [ ] Does this include contact information?
- [ ] Does this include medical, allergy, pastoral care, personnel, or financial details?
- [ ] Does this include passwords, tokens, API keys, or private account information?
- [ ] Is the organization comfortable using this data with an AI tool?
- [ ] Can I replace real data with fake or anonymized sample data?

If any answer creates concern, stop and use fake data for training.
