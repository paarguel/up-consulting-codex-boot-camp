# Date and Time Review Checklist

Use this for events, meetings, deadlines, and reports.

- [ ] Is the date present?
- [ ] Is the day of week consistent with the date?
- [ ] Is the start time present?
- [ ] Is the end time present?
- [ ] Is the time zone relevant?
- [ ] Is setup time different from event start time?
- [ ] Is cleanup time listed?
- [ ] Are deadlines clear?
- [ ] Are any dates inferred rather than sourced?

If Codex guessed or inferred a date, move it to Questions.
