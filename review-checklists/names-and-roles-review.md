# Names and Roles Review Checklist

Use this when assigning tasks, mentioning people, or creating contact lists.

- [ ] Names are spelled correctly.
- [ ] Roles are correct.
- [ ] Assignments match the source files.
- [ ] Unassigned tasks are clearly flagged.
- [ ] No one is assigned to conflicting roles.
- [ ] No private contact information is exposed unnecessarily.
- [ ] Any child or vulnerable-person information has been removed or handled according to policy.

When unsure, use a Questions section instead of guessing.
