# Sample Work Pack - Messy Roster Cleanup

## Scenario

A practice youth signup roster has duplicate rows, inconsistent formatting, and missing fields.

All data is fake.

## Goal

Use Codex to identify cleanup issues without inventing missing personal data.

## Suggested prompt

Use:

```text
prompt-recipes/roster-cleanup.md
```

## Expected outputs

```text
outputs/roster-cleanup-review.md
outputs/cleaned-roster.csv
```

Only create a cleaned CSV for safe formatting cleanup. Keep uncertain records in the review packet.
