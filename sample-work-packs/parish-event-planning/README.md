# Sample Work Pack - Parish Event Planning

## Scenario

The parish youth ministry team is planning a Friday youth night. The source materials are scattered across notes, volunteer availability, room setup details, a budget draft, and old parent email copy.

All data is fake.

## Goal

Use Codex to create:

- Event summary
- Timeline
- Volunteer assignments
- Supplies checklist
- Parent email draft
- Missing information list
- Risks or conflicts to review

## Suggested prompt

Use:

```text
prompt-recipes/event-command-center.md
```

## Expected output

```text
outputs/event-command-center.md
```

## Review risks

- Dates and times
- Room reservation
- Volunteer coverage
- Parent-facing tone
- Costs and supplies
- Any issue involving minors
