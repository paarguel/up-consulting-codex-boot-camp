# Sample Work Pack - Weekly Reporting

## Scenario

A ministry lead has scattered notes and needs a concise weekly leadership report.

All data is fake.

## Goal

Use Codex to create:

- Executive summary
- Wins
- Blockers
- Metrics
- Follow-up list
- Risks
- Next week priorities

## Suggested prompt

Use:

```text
prompt-recipes/weekly-report.md
```

## Expected output

```text
outputs/weekly-leadership-report.md
```
