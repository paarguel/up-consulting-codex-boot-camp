# Current Process Notes

Process: room request intake

Current rough flow:

1. Request comes by email or phone.
2. Office checks parish calendar.
3. If room is free, pencil it in.
4. Ask facilities if setup is needed.
5. Send confirmation email.
6. Add notes about tables, chairs, AV, and cleanup.

Problems:

- Sometimes setup notes are missing.
- Sometimes a room gets penciled in but not confirmed.
- Sometimes two ministries ask for the same room.
- Cleanup responsibilities are unclear.

Questions:

- Who approves outside groups?
- Who confirms insurance requirements?
- When does pastor approval apply?
