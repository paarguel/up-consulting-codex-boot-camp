# Sample Work Packs

These folders are fake but realistic practice materials for knowledge-worker labs.

## Work packs

1. `parish-event-planning` - turn messy event notes into a command center.
2. `meeting-to-action-items` - turn notes into minutes, decisions, and follow-ups.
3. `messy-roster-cleanup` - find duplicates, missing fields, and review risks.
4. `announcement-rewrites` - rewrite one announcement for several channels.
5. `weekly-reporting` - create a concise leadership report from scattered notes.
6. `vendor-comparison` - compare options against requirements and risks.
7. `sop-builder` - turn informal notes into an SOP and checklist.

## Standard pattern

Each work pack uses:

```text
inputs/
outputs/
workflow.md
```

Use the matching file in `prompt-recipes/` when running a lab.

## Safety

All data in these work packs is fake. Do not replace it with real sensitive data during training.
