# Sample Work Pack - Meeting to Action Items

## Scenario

A parish operations meeting produced scattered notes, unclear decisions, and several follow-up tasks.

All data is fake.

## Goal

Use Codex to create a meeting follow-up package:

- Summary
- Decisions
- Action tracker
- Open questions
- Follow-up email draft

## Suggested prompt

Use:

```text
prompt-recipes/meeting-minutes-action-tracker.md
```

## Expected output

```text
outputs/meeting-follow-up-package.md
```
