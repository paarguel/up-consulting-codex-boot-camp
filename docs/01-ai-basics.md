# 01 - AI Basics for Knowledge Workers

## Goal

Give the learner a simple mental model before asking them to use Codex.

They do not need a technical lecture. They need enough grounding to understand why AI can be useful, why it can be wrong, and why review matters.

## Short explanation

AI tools like ChatGPT and Codex are good at finding patterns, using context, drafting text, organizing information, and following instructions.

They are not good enough to be trusted blindly with truth, privacy, money, people, or final judgment.

## Whiteboard story

Use this simple progression:

```text
Search engine -> ChatGPT -> Custom GPTs and connectors -> Codex
```

### Search engine

You ask for pages and links.

The human has to open, read, compare, and turn the information into a useful artifact.

### ChatGPT

You ask in conversation.

The model can draft, explain, summarize, brainstorm, and rewrite. It is helpful, but it usually does not know your full work context unless you provide it.

### Custom GPTs, plugins, and connectors

You add instructions, tools, or connections to specific sources.

The assistant becomes more useful because it has a clearer job and better access to context.

### Codex

Codex works around a project folder.

For knowledge workers, that folder is the desk:

```text
inputs/
outputs/
workflows/
```

Codex can read the files, produce artifacts, explain what changed, run commands when needed, and help save repeatable workflows.

## Key vocabulary

### Model

The AI system that generates the response.

### Prompt

The instructions you give it.

### Context

The material it can use: files, notes, examples, constraints, tone guides, prior drafts, and the conversation.

### Tool use

When Codex does more than answer in chat, such as reading files, writing a document, checking a folder, or running a command.

### Hallucination

When the AI produces something that sounds confident but is unsupported, wrong, invented, or mixed up.

### Review

The human step where you check accuracy, privacy, tone, judgment, and whether the output is safe to use.

## The beginner mindset

Do not ask:

```text
Can AI do my job?
```

Ask:

```text
Which messy part of my work can Codex help me organize into a draft I can review?
```

## Teaching line

Use this sentence often:

> Codex is useful because it can turn messy inputs into a clear first draft. It is safe when we review the draft before trusting it.
