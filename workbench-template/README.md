# Workbench Template

Copy this folder when a learner wants to build a real workflow after the boot camp.

```text
workbench-template/
  inputs/
  outputs/
  workflows/
```

## How to use

1. Put source material in `inputs/`.
2. Ask Codex to inspect the folder before drafting.
3. Save generated artifacts in `outputs/`.
4. Save the repeatable process in `workflows/`.
5. Run the relevant review checklist before sending or publishing anything.

## Starter prompt

```text
Read the files in inputs/.

First, explain what materials are here and what output they seem suited for.

Then ask me up to five clarification questions before drafting.

Do not edit or create files yet.
```
