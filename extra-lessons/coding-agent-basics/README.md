# Extra Lessons - Coding Agent Basics

This folder preserves the original coding-focused course material.

Use it after the main knowledge-worker boot camp if a learner wants to understand how Codex helps with code.

## What this addendum teaches

- Inspecting a code folder
- Reading tests
- Making the smallest useful code change
- Running tests
- Reviewing diffs
- Understanding Git commits

## Start here

```text
exercises/README.md
```

## Important distinction

The main boot camp is for knowledge work:

```text
messy inputs -> clear output -> repeatable workflow
```

This addendum is for code work:

```text
inspect -> plan -> edit -> test -> review -> commit
```
