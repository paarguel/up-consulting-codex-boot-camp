# Prompt Recipe - Weekly Report

Use with:

```text
sample-work-packs/weekly-reporting/
```

## Prompt

```text
Use the work pack at sample-work-packs/weekly-reporting/.

Read everything in sample-work-packs/weekly-reporting/inputs/.

Create a weekly leadership report with:
1. executive summary
2. wins
3. blockers
4. metrics or attendance notes
5. follow-up list
6. risks or decisions needed
7. next week priorities

Rules:
- Separate facts from interpretation.
- Do not invent numbers.
- If a metric is unclear, put it in Questions.
- Keep the report concise and leader-friendly.
- Save the result in sample-work-packs/weekly-reporting/outputs/weekly-leadership-report.md.

After saving, list what the sender should verify before sharing.
```

## Review focus

- Numbers
- Names
- Status claims
- Commitments
- Audience sensitivity
