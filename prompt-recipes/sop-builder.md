# Prompt Recipe - SOP Builder

Use with:

```text
sample-work-packs/sop-builder/
```

## Prompt

```text
Use the work pack at sample-work-packs/sop-builder/.

Read everything in sample-work-packs/sop-builder/inputs/.

Turn the informal process notes into a standard operating procedure.

Include:
1. purpose
2. when to use this SOP
3. roles and responsibilities
4. step-by-step procedure
5. common mistakes
6. escalation or approval points
7. new-staff checklist
8. manager review checklist

Rules:
- Do not invent policy.
- If a step is unclear, put it in Questions.
- Keep the writing plain and practical.
- Save the result in sample-work-packs/sop-builder/outputs/standard-operating-procedure.md.

After saving, list which parts require manager approval before use.
```

## Review focus

- Policy accuracy
- Approval steps
- Role ownership
- Missing exceptions
- Training clarity
