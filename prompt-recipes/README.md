# Prompt Recipes

Prompt recipes are reusable starting points for the sample work packs.

## Recipes

- `event-command-center.md`
- `meeting-minutes-action-tracker.md`
- `roster-cleanup.md`
- `announcement-rewriter.md`
- `weekly-report.md`
- `vendor-comparison.md`
- `sop-builder.md`

## How to use

1. Open the matching sample work pack.
2. Ask Codex to inspect the inputs first.
3. Paste the recipe prompt.
4. Review the output with the relevant checklist.
5. Save improvements to the work pack's `workflow.md`.
