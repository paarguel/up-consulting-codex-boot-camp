# Prompt Recipe - Announcement Rewriter

Use with:

```text
sample-work-packs/announcement-rewrites/
```

## Prompt

```text
Use the work pack at sample-work-packs/announcement-rewrites/.

Read everything in sample-work-packs/announcement-rewrites/inputs/.

Rewrite the announcement for these channels:
1. bulletin
2. pulpit announcement
3. email
4. website
5. social media caption

Rules:
- Preserve the factual details from the source files.
- Do not add dates, times, costs, registration links, or contact names that are not in the inputs.
- Keep the tone warm, clear, and parish-office friendly.
- Put missing or unclear details in a Questions section.
- Save the result in sample-work-packs/announcement-rewrites/outputs/announcement-versions.md.

After saving, list what must be checked before publishing.
```

## Review focus

- Dates and times
- Registration details
- Contact person
- Tone by channel
- Missing approvals
