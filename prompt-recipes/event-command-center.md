# Prompt Recipe - Event Command Center

Use with:

```text
sample-work-packs/parish-event-planning/
```

## Prompt

```text
Use the work pack at sample-work-packs/parish-event-planning/.

Read everything in sample-work-packs/parish-event-planning/inputs/.

Create an event command center for this parish youth night.

Include:
1. one-paragraph event summary
2. timeline
3. volunteer assignment table
4. supplies checklist
5. parent email draft
6. missing information / questions
7. risks or conflicts to review

Rules:
- Do not invent missing details.
- If dates, names, costs, rooms, or assignments are unclear, put them in Questions.
- Keep the tone warm, organized, and parish-office friendly.
- Use only the files in this work pack.
- Save the result in sample-work-packs/parish-event-planning/outputs/event-command-center.md.

After saving, summarize what you created and list what a human must review before sending anything.
```

## Review focus

- Date and time
- Room reservation
- Volunteer assignments
- Parent-facing language
- Costs and supplies
- Missing approvals
- Anything involving minors
