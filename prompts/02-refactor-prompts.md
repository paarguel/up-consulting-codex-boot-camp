# Refactor Prompts

## Understand first

```text
Read this file and summarize what it does. Do not refactor yet.
```

## Identify smells

```text
List the top 3 code smells in this file. For each, explain why it matters.
```

## Refactor plan

```text
Propose a small refactor plan that preserves behavior. Include what tests or manual checks should be run.
```

## Apply one step

```text
Apply only step 1 of the refactor plan. Keep behavior the same.
```

## Confirm behavior

```text
Explain how we can confirm behavior stayed the same.
```
